# GoGoAnimeAPI-Bot
A Simple Telegram bot based on Telethon&amp;Pyrogram for Scraping Anime.

## How To Host
The easiest way to deploy this Bot
Enter your ```API_ID```,```API_HASH``` And ```TOKEN```.
